# 6. Pila Tecnológica y Bitácora de Avances

En esta sección se detallan las herramientas y tecnologías elegidas para el desarrollo del "Sistema de Gestión y Catálogo Web para Emprendimientos", así como un registro cronológico de las actividades ejecutadas a lo largo del ciclo de vida del proyecto.

---

## 6.1 Pila Tecnológica (Tech Stack)

Para garantizar la mantenibilidad, escalabilidad y rapidez en el desarrollo, el sistema se construirá utilizando las siguientes tecnologías:

*   **Backend (Lógica y Servidor):** 
    *   **Python:** Lenguaje principal del servidor, elegido por su legibilidad y robustez.
    *   **Flask:** Micro-framework web de Python. Proporciona las herramientas necesarias para construir aplicaciones web ligeras, manejar rutas (routing) y crear la API REST de forma rápida sin la sobrecarga de frameworks más grandes.
*   **Frontend (Interfaz de Usuario):**
    *   **HTML5 & CSS3:** Para la estructura semántica y el diseño responsivo del catálogo y panel de administración.
    *   **JavaScript (Vanilla JS):** Para la manipulación dinámica del DOM, validaciones en el lado del cliente y comunicación asíncrona (AJAX/Fetch) con el backend.
*   **Base de Datos:**
    *   **SQLite / PostgreSQL:** Base de datos relacional para el almacenamiento estructurado de usuarios, productos y ventas. SQLite para el entorno de desarrollo y PostgreSQL como opción para el despliegue en producción.
*   **Control de Versiones y Gestión de Código:**
    *   **Git:** Sistema de control de versiones local para el seguimiento de cambios en el código fuente.
    *   **GitHub:** Repositorio en la nube para el respaldo del proyecto, trabajo colaborativo y control de ramas (branches).
*   **Asistencia y Aceleración de Desarrollo:**
    *   **Antigravity (IA):** Entorno avanzado de programación asistida por Inteligencia Artificial para la estructuración de la arquitectura, autocompletado de código complejo, generación de la documentación técnica y refactorización continua.
