# Sistema de Gestión y Catálogo Web para Emprendimientos

Este proyecto consiste en el desarrollo de un Sistema de Gestión y Catálogo Web destinado a emprendimientos.

## Estructura del Proyecto

*   `docs/`: Documentación del proyecto.
*   `ai/`: Prompts y conversaciones con IA.
*   `src/`: Código fuente del proyecto.
*   `deliverables/`: Entregables generados.
